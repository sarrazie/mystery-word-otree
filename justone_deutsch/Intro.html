Sie wurden zu viert in Gruppen eingeteilt. Zusammen in Ihrer Gruppe spielen Sie insgesamt 12 Runden.

Ziel: Einer der Spieler – der Ratende – muss ein geheimes Wort anhand der Hinweise der anderen Spieler erraten.

Ein Hinweis besteht nur aus einem einzigen Wort.

Wenn zwei oder mehr Spieler dasselbe Wort schreiben, werden alle identischen Wörter entfernt, bevor der Ratende sie sieht.

{{ if player.participant.treatment == 3}} Sie können den Bonus erhalten, wenn Ihre Gruppe die beste Punktzahl aller Gruppen in dieser Sitzung erreicht. {{ else }} {{ endif }}

{{ if player.participant.treatment == 2}} Sie können den Bonus erhalten, wenn Ihre Gruppe die meisten Ideen für Hinweise in dieser Sitzung entwickelt. {{ else }} {{ endif }}

{{ if player.participant.treatment == 4}} Sie können den Bonus erhalten, wenn Ihre Gruppe die exklusivsten und einzigartigsten Hinweise in dieser Sitzung gibt. {{ else }} {{ endif }}

Jede Runde ist in vier Phasen unterteilt:

  1. Allen Spielern außer dem Ratenden wird mitgeteilt, für welches geheime Wort sie einen Hinweis geben müssen.
  2. Ohne miteinander zu kommunizieren und ohne es jemandem zu zeigen, gibt jeder Hinweisgeber einen Hinweis ein.
  3. Sobald alle Spieler ihre Hinweise geschrieben haben, werden identische oder ungültige Hinweise entfernt. Der Ratende sieht alle verbleibenden Hinweise und hat nur einen Versuch, um das geheime Wort zu erraten.
  4. Der Gruppe wird mitgeteilt, ob der Ratende das geheime Wort korrekt erraten hat oder nicht. Die nächste Runde beginnt, und ein anderer Spieler wird der Ratende.
