Sie wurden zu viert in Gruppen eingeteilt. Zusammen in Ihrer Gruppe spielen Sie insgesamt 12 Runden.

Ziel: Einer der Spieler – der Ratende – muss ein geheimes Wort anhand eines Hinweispaars der anderen Spieler erraten.

Ein Hinweispaar besteht aus zwei einzelnen Wörtern.

Wenn zwei oder mehr Gruppen ein identisches Hinweispaar schreiben, wird das Hinweispaar entfernt, bevor der Ratende es sieht, und die Runde ist beendet.

{{ if player.participant.treatment != 1}} Ihre Auszahlung besteht aus einer Teilnahmegebühr von 3,50€ und einem Bonus von bis zu 8€, abhängig von Ihrer Leistung. {{ else }} Sie erhalten 10€, wenn Sie an allen 12 Runden teilnehmen. {{ endif }}

{{ if player.participant.treatment == 3}} Sie können den Bonus erhalten, wenn Ihre Gruppe die beste Punktzahl aller Gruppen in dieser Sitzung erreicht. {{ else }} {{ endif }}

{{ if player.participant.treatment == 2}} Sie können den Bonus erhalten, wenn Ihre Gruppe die meisten Ideen für Hinweispaare in dieser Sitzung entwickelt. {{ else }} {{ endif }}

{{ if player.participant.treatment == 4}} Sie können den Bonus erhalten, wenn Ihre Gruppe die exklusivsten und einzigartigsten Hinweispaare in dieser Sitzung gibt. {{ else }} {{ endif }}

Jede Runde ist in sechs Phasen unterteilt:

  1. Allen Spielern außer dem Ratenden wird mitgeteilt, für welches geheime Wort sie ein Hinweispaar geben müssen.
  2. Ohne miteinander zu kommunizieren generiert zunächst jeder Hinweisgeber passende Hinweispaare.
  3. Sobald alle Spieler Hinweispaare geschrieben haben, kommen die Hinweisgeber zusammen und kommentieren ihre bisher geschriebenen Hinweispaare.
  4. Danach hat jeder Hinweisgeber nochmals Zeit, seine Hinweispaare zu überarbeiten.
  5. In der vorletzten Phase bestimmen die Hinweisgeber nun gemeinsam, welches eine Hinweispaar sie an den Ratenden weitergeben wollen.
  6. Wenn das Hinweispaar ungültig oder identischen zu einer anderen Gruppe ist, wird es entfernt. Der Ratende sieht alle verbleibenden Hinweise und hat nur einen Versuch, um das geheime Wort zu erraten.

Der Gruppe wird nach der Runde mitgeteilt, ob der Ratende das geheime Wort korrekt erraten hat oder nicht. Die nächste Runde beginnt, und ein anderer Spieler wird der Ratende.
