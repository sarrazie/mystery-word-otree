Invalid Clue Pair:

  • The clue pair contains more than one single word per clue.
    • Example: The clue pair "very dangerous" + "green" is not allowed since it contains more than two words in total.
  • The clue pair contains the mystery word in a different spelling.
    • Example: Pyjamas is not allowed if the mystery word is Pajamas.
  • The clue pair contains the mystery word in another language.
    • Example: Schwarz is not allowed if the mystery word is Black.
  • The clue pair contains a word with the same root as the mystery word.
    • Example: Princess is not allowed if the mystery word is Prince.
  • The clue pair contains a made-up word.
    • Example: Sweetthing is not allowed if the mystery word is cake.
  • The clue pair contains a clue with special characters.
    • Example: #sweet* is not allowed if the mystery word is chocolate.
